# AI Script Templates

This directory contains example template files for Frida and Ghidra scripts used in Intellicrack's reverse engineering operations.

## Available Templates:

- **frida_license_bypass.js** - Example Frida script template for license bypass operations with placeholders for dynamic content
- **ghidra_analysis.py** - Example Ghidra script template for binary analysis with placeholders for dynamic content

## Template Structure:

These files serve as reference templates showing the structure and placeholders used for dynamic script generation. The current Intellicrack AI system generates scripts using embedded template logic rather than loading these files directly.

## Placeholders:

Templates contain placeholders like `{{target_binary}}`, `{{protection_types}}`, `{{script_name}}`, etc., which are replaced with actual values during script generation.
