# Intellicrack Protection Engine

The Intellicrack Protection Engine is a powerful binary protection detection system
integrated into the Intellicrack framework.

## Features

- Comprehensive protection detection
- Support for multiple file formats (PE, ELF, APK, etc.)
- Extensive signature database
- Command-line and GUI interfaces

## Usage

Command-line:
```
icp-engine.exe <filename>
```

GUI:
```
icp-gui.exe
```

## Signature Files

Signature files are stored in the `signatures/` directory with the `.ics` extension.
